# TimeShare - 세대공생 시간은행 플랫폼

![TimeShare Logo](https://via.placeholder.com/800x200/C9965F/FFFFFF?text=TimeShare)

세대 간 재능과 시간을 나누는 혁신적인 플랫폼입니다. 청년과 시니어가 서로의 재능을 교환하며 세대 간 연결을 만들어갑니다.

## 🌟 주요 기능

### 📱 이중 UI 모드
- **청년 모드**: 효율적이고 세련된 인터페이스 (14-16px 폰트)
- **시니어 모드**: 크고 명확한 UI (18-22px 폰트, 큰 터치 영역)
- 설정에서 언제든 모드 전환 가능

### ⏰ 시간은행 시스템
- 타임 기반 거래 시스템
- 1일 최대 4시간 거래 제한
- 세대 교류 시 +0.5시간 보너스
- 15타임부터 지역화폐로 전환 가능 (1타임 = 1,000원)

### 🎯 핵심 기능
1. **게시글 작성 및 검색**
   - 제공/요청 타입 선택
   - 6가지 카테고리 (교육, 돌봄, 생활지원, IT/기술, 언어교환, 예술/문화)
   - 지역 및 시간대 필터링

2. **인증 시스템**
   - 신원 인증 (휴대폰 본인 인증)
   - 재능 인증 (재학/재직증명서, 자격증, 이력서)
   - 인증 배지 표시

3. **이력서 기능**
   - 재능 인증 완료 시 공개 가능
   - 보유 기술, 경력, 거래 이력 표시
   - 받은 평가 모음
   - 다운로드 및 공유 기능

4. **매칭 시스템**
   - AI 분석 로딩 애니메이션
   - 실시간 채팅 (시뮬레이션)
   - 거래 시작 및 완료
   - 상호 평가 시스템

5. **친구 추천 시스템**
   - 추천 코드 생성 및 공유
   - 추천인과 가입자 모두 +1시간 보너스

## 📊 더미 데이터

### 사용자 (20명)
- 청년층 10명 (20-35세)
- 시니어층 10명 (60-75세)
- 다양한 인증 상태 및 재능 보유

### 게시글 (30개)
- 6가지 카테고리에 골고루 분포
- 제공/요청 타입 혼합
- 세대 교류 가능한 게시글 포함

## 🔑 주요 플로우

### 1. 신규 사용자
1. 모드 선택 (청년/시니어)
2. 회원가입
3. 신원 인증
4. 메인 화면

### 2. 게시글 작성
1. 게시글 작성 클릭
2. 재능 인증 요구 (미인증 시)
3. 재능 인증 완료
4. 게시글 작성 및 등록

### 3. 거래 진행
1. 게시글 검색 및 선택
2. 매칭 신청
3. 세대 확인 및 보너스 안내
4. 채팅 및 거래 시작
5. 거래 완료 및 타임 정산
6. 상호 평가

### 4. 지역화폐 전환
1. 마이페이지 접속
2. 지역화폐 전환 섹션
3. 15타임 이상 보유 확인
4. 전환 신청 (1타임 = 1,000원)

## 💡 특별 기능

### 세대 교류 보너스
- 다른 세대와 거래 시 자동으로 +0.5시간 보너스 적용
- 게시글 카드에 세대 교류 배지 표시
- 상세 페이지에서 보너스 안내

### 1일 거래 제한
- 하루 최대 4시간까지 거래 시작 가능
- 상단 바에서 남은 제한 시간 확인
- 진행 중인 거래는 계속 가능
- 프리미엄 멤버십으로 제한 해제

### 프리미엄 멤버십
- 월 4,900원
- 무제한 거래
- 우선 매칭
- 특별 배지

## 🎯 주요 페이지

### 모드 선택
- 최초 진입 시 표시
- 청년/시니어 모드 선택
- 각 모드의 특징 설명

### 메인 화면
- 타임 정보 카드 (보유 타임, 오늘 사용 가능)
- 검색바 및 카테고리 탭
- 게시글 리스트
- 하단 네비게이션 (홈, 작성, 매칭, MY)

### 게시글 작성
- 제공/요청 선택
- 카테고리 및 상세 정보 입력
- 필요 타임 설정 (1-15)
- 가능한 시간대 선택
- 1일 제한 안내

### 상세 페이지
- 게시글 정보
- 작성자 프로필 및 평점
- 세대 교류 보너스 표시
- 이력서 보기 (재능 인증 시)
- 유사 게시글 추천

### 마이페이지
- 프로필 및 인증 상태
- 타임 정보 및 사용 내역
- 지역화폐 전환
- 친구 초대
- 인증 센터
- 프리미엄 멤버십

### 재능 인증
- 인증 유형 선택 (학생, 직장인, 전문가, 경력자)
- 서류 업로드 (JPG, PNG, PDF)
- 인증 상태 확인

### 이력서
- 기본 정보 및 보유 기술
- 경력 사항
- 거래 이력 및 통계
- 받은 평가 모음
- 다운로드 및 공유

### 매칭 시스템
- AI 분석 로딩
- 매칭 성공 및 상대방 정보
- 실시간 채팅
- 거래 시작 및 완료
- 사용 시간 조정
- 상호 평가
